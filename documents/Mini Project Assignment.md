Mini Project Assignment 
Design and Development of a Complete Database System for 
a Forensic Medical Department 
For a Forensic Medicine Department complete database system project, a group of 4 students is 
ideal. 
1. Background 
Currently, many forensic medical departments maintain records manually using paper-based 
files. This creates difficulties in: 
• 
Searching patient/case records 
• 
Maintaining confidentiality 
• 
Generating court reports 
• 
Tracking evidence and samples 
• 
Managing medico-legal documents 
• 
Producing statistical reports 
Students are required to analyze the existing manual process and develop a complete 
computerized database system for a forensic medical department. 
 
2. Project Objectives 
The objectives of this project are to: 
• 
Understand real-world database design 
• 
Practice requirement gathering and system analysis 
• 
Design conceptual, logical, and physical database models 
• 
Implement a relational database 
• 
Develop a user-friendly application interface 
• 
Generate medico-legal reports 
• 
Apply normalization and database security concepts 
3. Expected System Modules 
Each group should develop a complete database system including the following modules. 
Core Modules 
1. Patient Management 
o Register patients 
o Maintain personal details 
o Track patient history 
2. Case Management 
o Create medico-legal cases 
o Assign case numbers 
o Record incident details 
3. Postmortem Management 
o Record autopsy details 
o Store cause of death 
o Maintain examination reports 
4. Evidence Management 
o Track forensic samples 
o Maintain chain of custody 
o Store laboratory evidence information 
5. Court Report Management 
o Generate medico-legal reports 
o Produce court submission documents 
o Track report status 
6. Staff Management 
o Doctors 
o Judicial Medical Officers 
o Laboratory staff 
o Clerical officers 
7. User Authentication and Security 
o Login system 
o User roles 
o Access control 
8. Report Generation 
o Daily case reports 
o Monthly statistics 
o Pending case reports
